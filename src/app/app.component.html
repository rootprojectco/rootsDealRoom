ROOTS Deal Rooms

Use a Web3 Supported Wallet

ROOTS Deals Room is a smart-contracts which allow users to participate in auctions for buying and selling ROOTS tokens. We use a standard protocol called Web3 to enable this service.

In order to be able to use your blockchain wallet (some refer to this as a MyEtherWallet account), you will need to use the MetaMask browser add-on.

Step 1: Download the Appropriate Browser

MetaMask currently only supports Chrome, Firefox or Opera and does not operate on mobile or tablets. You will need to have one of these browsers before we proceed.

Download Chrome | Download Firefox | Download Opera

Step 2: Install MetaMask

Click on the link below in order to download the MetaMask addon directly from their website.
